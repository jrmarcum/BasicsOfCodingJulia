Julia arrays are 1-indexed. 1-D arrays print with comma separators, e.g. `[0, 0, 0, 0, 0]`. 2-D arrays print in matrix notation, e.g. `[0 1 2; 1 2 3]`.

___

##### Run Command:
`julia arrays.jl`

##### Results:
`emp: [0, 0, 0, 0, 0]`
`set: [0, 0, 0, 0, 100]`
`get: 100`
`len: 5`
`dcl: [1, 2, 3, 4, 5]`
`2d:  [0 1 2; 1 2 3]`
