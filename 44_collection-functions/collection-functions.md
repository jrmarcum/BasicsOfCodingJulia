Julia has built-in `filter`, `map`, and `findfirst` that replace Go's manual collection utilities. Arrays print with brackets and commas rather than Go's bracket-only format.

___

##### Run Command:
`julia collection-functions.jl`

##### Results:
`2`
`false`
`true`
`false`
`["peach", "apple", "pear"]`
`["PEACH", "APPLE", "PEAR", "PLUM"]`
